# Проект «Buddy Light»

## Краткая суть

**«Buddy Light»** — это система волонтёрской поддержки иностранных студентов в первые дни обучения. Проект состоит из двух простых частей:

1. **Одностраничный сайт-визитка (лендинг)** с чек-листом, контактами волонтёров и анонсами мероприятий.
2. **Живые Buddy** — обычные студенты, которые помогают новичкам через Telegram без бюрократии.

---

## Проблема, которую решает проект

Иностранный студент в **первые 72 часа** испытывает:

-  **языковой шок** (не понимает объявления, расписание, меню в столовой);
-  **потерю в пространстве** (не может найти столовую, деканат, аптеку);
-  **страх просить помощь** (думает, что его отправят в деканат заполнять бумаги).

> Университетские сайты перегружены документами, а деканат отвечает медленно.  
> Новичку нужен **один живой контакт**, который ответит в Telegram за 5 минут на простой вопрос: *«А что значит "хвосты"?»*

---

## Целевая аудитория

| Тип аудитории      | Кто именно                                                                                                 |
| ------------------ | ---------------------------------------------------------------------------------------------------------- |
| **Прямая**         | Иностранные студенты (бакалавриат, магистратура, подготовительные курсы) — первые 1–4 недели после приезда |
| **Опосредованная** | Местные студенты-волонтёры (получают опыт межкультурной коммуникации, строят портфолио)                    |

# Структура сайта «Buddy Light»

## 1. Шапка (3 секунды на понимание)

- **Заголовок:** `«Buddy»`
- **Подзаголовок:** *«Твой знакомый студент, который покажет, где вкусно и не страшно»*
- **Кнопка:** `«Нужен Buddy»` → ведёт к контакту в Telegram

---

## 2. Блок «Кто такой Buddy?»

> Поможет дойти до столовой  
> Переведёт объявление у подъезда  
> Ответит на вопрос *«А что значит "автомат"?»*

**Одна фраза жирным:**  

> **«Это не служба, это просто друг»**

---

## 3. Главный блок: Чек-лист «Первые 3 дня»

### День 1

- [ ] Получить сим-карту (Buddy подскажет где)
- [ ] Показать карту кампуса
- [ ] Купить булочку без очереди (лайфхак)

### День 2

- [ ] Скачать приложение транспорта
- [ ] Найти свою аудиторию по номеру
- [ ] Понять, что такое «окно» в расписании

### День 3

- [ ] Сходить в ближайший магазин за гречкой (не перепутать с крупой для собак)
- [ ] Записаться в поликлинику (с переводчиком)

> **Кнопка внизу чек-листа:** «Скачать PDF-памятку» → ссылка на Google Docs или картинку

---

## 4. Ближайшее мероприятие

### Фотоохота «Секреты города»

- **Дата:** *27 апреля, 15:00*  
- **Место:** *Сбор у центрального входа в главный корпус*  

**Кнопка:** «Пойду, дайте Buddy» → ссылка на Telegram-чат мероприятия
